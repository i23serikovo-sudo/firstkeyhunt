В данной задаче вам потребуется найти все части ключа спрятанные по файловой системе.

СТРУКТУРА ЗАДАНИЯ:
 В этой директории ( ~ ) находится пять папок в которых будет каким либо образом спрятан файл key.txt с куском ключа.
 Ключ собирается по порядку директорий в одну строку в файл answer.txt .
 Когда вы соберете полный ключ, вы можете проверить свое решение с помощью python-скрипта test.py выполнив команду

 python test.py

 если вы все сделали верно то на выходе получите строку на русском языке которая укажет 
 что нужно сделать чтобы получить приз.
